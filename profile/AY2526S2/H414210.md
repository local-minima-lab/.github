title: "ExpOps: Declarative Orchestration for ML Experiments"
author: Poon Zhe Xuan
date: 2026-04-20 23:00:00 +0800
---

ExpOps (Experiment Operations) is a platform for running machine learning experiments without the usual orchestration headaches. You describe your pipeline once in a YAML config, write your stages as regular Python functions and ExpOps takes care of the rest: figuring out the execution order, running stages in parallel, caching results that haven't changed and scaling the same pipeline from your laptop up to a compute cluster with no code changes.

The goal is to let you focus on the experiment instead of the orchestration. If you tweak one hyperparameter or code for one process, only the affected stages re-run. Random seeds are handled for you across every worker, so the same pipeline gives the same results whether you or a teammate runs it. A companion web app with a visual pipeline builder and a live run monitor lets you see what's happening during long runs. Plugin component libraries are also implemented to provide a drag-and-drop no-code ML pipeline creator for beginners.
